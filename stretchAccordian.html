required | optional | not used

max - datatype:number, default:N/A, The width or height (in pixels) of a fully expanded kwick element. If isVertical is true, then max refers to the height - otherwise it is the width.
min - datatype:number, default:N/A, The width or height (in pixels) of a fully collapsed kwick element. If isVertical is true, then min refers to the height - otherwise it is the width.
isVertical - datatype:boolean, default:false, Kwicks will align vertically if true.
sticky - datatype:boolean, default:false, One kwick will always be expanded if true.
defaultKwick - datatype:number, default:0, The initially expanded kwick (if and only if sticky is true). Note: Zero based, left-to-right (or top-to-bottom).
event - datatype:boolean, default:'mouseover', The event that triggers the expand effect. Other likely candidates are click and dblclick.
duration - datatype:number, default:200, The number of milliseconds required for each animation to complete.
easing - datatype:boolean, default:N/A, A custom easing function for the animation (requires easing plugin).
spacing - datatype:number, default:0, The width (in pixels) separating each kwick element.
